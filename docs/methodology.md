# 방법론 노트

## 1. 프로젝트 목적

뉴스 감성 신호를 다음 거래일 포지션으로 연결하고 비용 차감 성과를 계산한다.

## 2. 핵심 질문

단순 감성 신호가 비용을 차감한 뒤에도 방향성 전략으로 의미 있는가?

## 3. 입력 데이터

- 파일: `data/sample/news_signal_returns.csv`
- 성격: 구조 검증용 합성 샘플
- 목적: 계산 흐름과 결과 해석을 작은 표본으로 먼저 검증

## 4. 계산 절차

1. 감성 label을 long/short/flat 포지션으로 변환하고 다음날 수익률에 적용한다.
2. 포지션 변경 시 거래비용을 차감해 look-good bias를 줄인다.
3. 뉴스 문장과 수익률은 신호 생성과 성과 연결을 설명하는 합성 데이터다.

## 5. 산출물 해석

`outputs/tables/baseline_results.csv`는 프로젝트별 baseline 결과입니다. 이 표는 최종 연구 결론이 아니라, 이후 실제 데이터 연결과 상세 분석을 시작하기 위한 구조 검증용 산출물입니다.

## 6. 한계

- 합성 데이터 기반이므로 통계적 유의성이나 투자 가능성을 주장하지 않습니다.
- 거래비용, 세금, 슬리피지, 유동성 제약은 일부 프로젝트에서만 단순 반영했습니다.
- 프로젝트별 정교한 모델링은 별도 후속 작업에서 확장해야 합니다.
